# 洞洞板画图助手 2.0 使用说明

## 功能介绍
洞洞板画图助手是一个用于绘制洞洞板电路连线的工具，支持以下功能：

- **网格绘制**：自动生成洞洞板网格，支持自定义行数和列数
- **连线绘制**：支持绘制红色和蓝色两种颜色的连线
- **元器件添加**：可以在连线上添加元器件并标注文字
- **方形绘制**：支持绘制方形区域
- **缩放和平移**：支持鼠标滚轮缩放和中键平移
- **撤销功能**：支持 Ctrl+Z 撤销操作
- **保存功能**：支持保存为图片和存档文件

## 安装说明

### 方法一：直接运行可执行文件
1. 打开 `洞洞板画图助手软件` 文件夹
2. 双击 `洞洞板画图.exe` 即可运行

### 方法二：使用 Python 运行
1. 确保已安装 Python 3.10 或更高版本
2. 安装依赖：`pip install pillow`
3. 运行 `pegboard_drawing.py` 文件

## 使用方法

### 基本操作
- **设置行列数**：在顶部输入框中输入行数和列数，点击「应用」按钮
- **绘制连线**：点击两个点即可绘制连线
- **切换颜色**：按 Tab 键切换连线颜色（红色/蓝色）
- **绘制元器件**：按住 Shift 键点击两个点
- **绘制方形**：按住 Alt 键点击两个点，再点击第三个点完成方形
- **编辑元器件文字**：双击元器件可编辑文字
- **删除选中**：点击「删除选中」按钮删除当前选中的元素
- **清空所有**：点击「清空」按钮清空所有绘制内容

### 高级操作
- **缩放**：鼠标滚轮上下滚动
- **平移**：按住鼠标中键拖动
- **撤销**：按 Ctrl+Z 撤销上一步操作
- **保存图片**：点击「保存图片」按钮将当前画布保存为图片
- **保存存档**：点击「保存存档」按钮将当前状态保存为存档文件
- **加载存档**：点击「加载存档」按钮加载之前的存档

## 界面说明

### 顶部控制面板
- **行数/列数输入框**：设置洞洞板的行数和列数
- **应用按钮**：应用新的行列数设置
- **设置按钮**：打开设置窗口，可设置保存路径等
- **保存图片/存档按钮**：保存当前画布或状态
- **加载存档按钮**：加载之前的存档
- **删除选中/清空按钮**：删除选中元素或清空所有内容
- **颜色和模式显示**：显示当前的颜色和绘制模式

### 画布区域
- **网格点**：黑色圆点表示洞洞板的孔洞
- **列号**：顶部显示列号（从右往左 1, 2, 3...）
- **行号**：左侧显示行号（从下往上 A, B, C...）
- **连线**：红色或蓝色线条表示电路连线
- **元器件**：带有文字标注的元件
- **方形**：淡绿色的方形区域

## 常见问题

### Q: 如何调整洞洞板大小？
A: 在顶部输入框中输入新的行数和列数，点击「应用」按钮。

### Q: 如何更改连线颜色？
A: 按 Tab 键可以在红色和蓝色之间切换。

### Q: 如何保存我的作品？
A: 点击「保存图片」按钮可以保存为图片，点击「保存存档」按钮可以保存为存档文件。

### Q: 如何撤销操作？
A: 按 Ctrl+Z 键可以撤销上一步操作。

### Q: 如何添加元器件？
A: 按住 Shift 键，点击两个点即可添加元器件。
